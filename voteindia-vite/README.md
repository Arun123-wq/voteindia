# VoteIndia — Online Voting System

## Setup
```bash
npm install
npm run dev
```
Open http://localhost:5173

## Demo Flow
1. Click **Register** → complete 4 steps
2. Click **Login** → any input → Send OTP → Verify & Login
3. Click **Vote** → pick candidate → confirm
4. View **Dashboard** and **Voter Card**

## Pages
- Home, Login, Register, Elections, Parties, Vote, Results, Dashboard, Voter Card, About
